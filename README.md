# Simple-Earthquake-Location
This demonstrates a simple example of earthquake location, allowing to experience how station and phase picking errors affect absolute location. The differential time approach can largely eliminate these effects <br>
---
- Compared to formal location methods, it is more like a simulation and test <br>
